# 🌷 Projeto O Fascínio das Tulipas

Bem-vindo ao repositório do projeto **O Fascínio das Tulipas**! Esta é uma aplicação web estática, responsiva e interativa desenvolvida para celebrar a beleza, a história e os cuidados associados a uma das flores mais populares do mundo.

---

## 🎯 Objetivo do Projeto
O principal objetivo deste projeto é criar uma página web informativa, visualmente atraente e de alta usabilidade sobre as tulipas. O site serve tanto como uma enciclopédia digital simplificada quanto como um guia prático, centralizando dados históricos (como a Tulipomania), o significado cultural de suas cores, instruções de cultivo e recursos multimídia.

---

## 🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web moderna, sem a necessidade de frameworks externos, priorizando a performance e a semântica:

* **HTML5:** Utilizado para a estruturação semântica do conteúdo (tags como `<header>`, `<section>`, `<main>` e `<footer>`), garantindo melhor acessibilidade e SEO.
* **CSS3:** Responsável por toda a identidade visual, paleta de cores temática (tons florais e pastéis), tipografia e design responsivo (adaptável para smartphones, tablets e desktops).
* **JavaScript (Vanilla):** Aplicado para adicionar interatividade na página, como manipulação do DOM, controle de componentes dinâmicos e melhorias na experiência do usuário.

---

## ✨ Funcionalidades
* **Layout Responsivo:** Adaptação fluida para qualquer tamanho de tela através de media queries e técnicas de design flexível.
* **Guia Interativo de Cores:** Seção dedicada ao significado das cores das tulipas com interações visuais.
* **Integração Multimídia:** Incorporação responsiva de players de vídeo (YouTube) para enriquecimento do conteúdo educativo.
* **Navegação Fluida:** Estrutura organizada em seções bem definidas que facilitam a leitura e escaneabilidade do texto.
